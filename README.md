# 📚 Online Book Borrowing Platform

# Name: PageSwap

A modern web application where users can explore, borrow, and manage books online with a smooth and user-friendly experience.

---

## 🚀 Live Demo

(https://assignment-008-azure.vercel.app/)

---

## 🛠️ Tech Stack

- **Frontend:** Next.js
- **Styling:** Tailwind CSS, DaisyUI
- **Authentication:** BetterAuth
- **Language:** JavaScript

---

## ✨ Features

- 🔐 User Authentication (Sign Up / Login / Logout)
- 👤 User Profile Management (Update Name & Image)
- 📖 Browse Available Books
- 📚 Borrow Books System
- 📦 Book Availability Tracking
- 🎯 Clean and Responsive UI
- ⚡ Fast Performance with Next.js

---

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# 1. Clone the repository
git clone https://github.com/AlFahadMunna/assignment-008.git

# 2. Go to project folder
cd your-repo-name

# 3. Install dependencies
npm install

# 4. Run development server
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the root directory and add:

```
NEXT_PUBLIC_BASE_URL=your_url
AUTH_SECRET=your_secret
```

_(Update based on your BetterAuth configuration)_
